ScheduleSite

ScheduleSite is a longer project of mine. It utilizes the Windesheim schedule API, And then requests the schedule for the currently logged in user.

This project took a lot longer then originally planned, due to difficulty in automatically retrieving the api key for the user. The solution was very complicated, and invoked the use of both browser cookies and a MongoDb collection.

Right now this site is online, and can actually be used by the students of Windesheim Almere.
